Superadmission is an early-stage infrastructure project focused on the coordination problem in Indian higher education admissions. It is a proposed coordination model — designed to address the gap between existing counselling systems and remove the operational burden currently carried by students navigating multiple systems simultaneously.

The project is being actively built by a small founding team. The work spans architecture design, workflow documentation, allocation logic simulation, and direct engagement with the problem at an operational level.

## What the Project Is Not

<Warning>
  These clarifications matter for any institutional or authority reader approaching this documentation.
</Warning>

| What it is not | What it is |
|----------------|-----------|
| A deployed system | A proposed coordination model under active development |
| A government initiative | An independent project seeking eventual institutional alignment |
| A counselling service | A workflow coordination layer, not a counselling system |
| A student-facing consumer app | An infrastructure architecture |
| An approved or certified system | A documented model, transparently public |
