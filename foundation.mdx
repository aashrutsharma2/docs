Students currently navigate multiple, disconnected counselling systems. They create separate accounts, upload identical documents across different portals, and manually track colliding deadlines.

Superadmission is a **proposed shared infrastructure** designed to unify these touchpoints. It is intended to function as a coordination layer that sits above existing allocation systems like JoSAA and State CETs, **without replacing them or altering how seats are allocated**.

## The Proposed Shift

Superadmission is designed to streamline the redundant steps in the admissions lifecycle. If implemented, the infrastructure would shift the administrative burden away from the student.

- **Unified Identity:** Instead of creating new accounts per counselling, students build **one profile** linked to standard identity frameworks.
- **Centralised Verification:** Rather than uploading the same documents repeatedly, files are fetched directly via **DigiLocker** and verified once.
- **Synchronised Tracking:** Manual deadline tracking is replaced by a **unified dashboard** offering side-by-side allotment comparisons and timeline alerts.
- **Accessible Guidance:** Premium consulting is replaced by **PraveshAI**, providing actionable, data-backed guidance to every participating student.

## India Stack Alignment

This architecture is designed to integrate seamlessly with India's existing digital public infrastructure. It acts as a shared protocol layer, much like UPI does for payments.

<CardGroup cols={2}>
  <Card title="Aadhaar Integration" icon="fingerprint">
    Intended identity layer utilising optional e-KYC and OTP-based authentication for secure, one-time profile creation.
  </Card>

  <Card title="DigiLocker" icon="file-shield">
    Proposed document layer to fetch mark sheets and rank cards directly from issuing authorities with explicit student consent.
  </Card>

  <Card title="UPI Gateway" icon="mobile">
    Designed payment layer to consolidate registration, counselling, and acceptance fees into a single interface.
  </Card>

  <Card title="APAAR" icon="id-card">
    Aligned with NEP 2020 to ensure persistent student records compatible with the Academic Bank of Credits.
  </Card>
</CardGroup>
