Superadmission is an early-stage infrastructure project focused on the coordination layer problem in Indian higher education admissions. It is not a product company. It is not a counselling service. It is a proposed public infrastructure model — designed to sit above existing counselling systems and reduce the operational burden currently carried by students.

The project is being built by a small founding team. The work at this stage is primarily architectural: designing systems, documenting workflows, simulating allocation logic, and engaging with the problem deeply enough to propose something credible.

## What the Project Is Not

These clarifications matter for any institutional reader approaching this documentation.

| What it is not | What it is |
|----------------|-----------|
| A deployed system | A proposed architecture |
| A government initiative | An independent project seeking eventual alignment |
| A counselling service | An infrastructure layer designed to sit above counselling |
| A student-facing consumer app | A workflow and coordination architecture |
| An approved or certified system | A documented model under development |
